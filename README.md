# XCharts
A Simple UGUI Charting Library for Unity

# 特性
  1. 支持折线图(`Line`)、柱状图(`Bar`)、饼图(`Pie`)、雷达图(`Radar`)等
  2. 支持`Default`、`Light`、`Dark`三种主题切换
  3. 参数可视化配置，效果实时预览
  4. 折线图通过参数可配置出：折线图、曲线图、区域图等
  5. 饼图通过参数可配置出：饼图、环形图、南丁格尔玫瑰图等
